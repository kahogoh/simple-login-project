# Simple Login Project

ES6 Style

* react
* react-router
* firebase
* webpack
* babel
* eslint
* esdoc

## setup
```
npm install
```

## run project
```
npm start
```
* project run at http://localhost:8080/

## documentation

* using ESDoc

```
npm run doc
```

* you can see the documentation at http://localhost:8080/doc/
